### Data Dictionary

1. Student Apptitude :
 - `student_id` = **int** *(unique pk)*
 - `course_level` = **str** *(this columns explains the student's course level)*

 - `aptitude_score` = **int** *(prediction test scores of students)* 

2. Student Performance :
 - `student_id` = **int** *(unique pk)*
 - `performance_score` = **int** *(test scores of students)*
